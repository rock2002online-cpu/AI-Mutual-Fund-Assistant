# InvestGPT – AI Investment Assistant

## Project Overview

InvestGPT is an AI-powered investment analysis platform that helps investors manage and analyze:

- Mutual Funds
- Stocks
- ETFs
- SIPs
- Portfolio Performance
- Financial Goals

The project is being built in Python from scratch as a learning journey and will gradually evolve into a fully automated investment assistant.

---

## Current Features

- ✅ Read portfolio from Excel
- ✅ Calculate investment amount
- ✅ Portfolio summary
